IMGCRY

2020

After reading about steganography - the practice of concealing a message within another message - I knew I had to try it myself. This program can hide a text by splitting and storing it as the least significant bit of an images' pixel colour value. The chages are very miniscule and usually it's impossible to tell the old and new picture apart.

3D Ortographic projection

2019

This program can draw and rotate any convex shape that consists of triangles. Written in pascal using graph library.
